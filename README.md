# Animal Protection Awareness System

A GUI-based Python application designed to spread awareness about animal protection
and allow users to report injured animals.

## Features
- View protected animals
- View threats to animals
- Protection guidelines
- Report injured animals
- Stores reports locally

## Technologies Used
- Python
- Tkinter (GUI)
- File Handling

## How to Run
1. Install Python
2. Clone or download the repository
3. Run: python main_gui.py

## Future Scope
- Web and mobile application
- Database integration
- Multi-language support
